# Reference letters

The **real signed reference letter images**, parsed from the
[Recommendations page](https://sites.google.com/view/margaritaaleeva/projects-recommendations)
of the portfolio.

The landing page does **not** show these as previews — the Recommendations cards use
text placeholders, and each card's **Download letter** button downloads the matching
image below (the `download` attribute saves it on click).

| File | Reference from |
| --- | --- |
| `yandex_reference_letter_scan.jpg` | Yandex (English) |
| `findguide_reference_letter_scan.jpg` | FindGuide |
| `fontanka_reference_letter_scan.jpg` | Fontanka.ru |
| `academic_reference_letter_scan.jpg` | Academic faculty |
| `copywriting_reference_letter_scan.jpg` | Copywriting lead |

The images are medium-resolution (captured from the live site). To upgrade, drop in a
higher-resolution scan using the **same filename** — no code change needed.
